이 문제는 브론즈 문제이다.

그럼에도 불구하고 글을 작성하는 이유는 함수 하나를 기억하기 위해서이다.

내 친구는 자바로 이 문제를 풀었는데

브론즈가 아닌 KMP알고림즘을 사용해서 골드정도의 난이도로 문제를 풀고 있는 것을 봤다.

C++ 사용자들을 해맑게 웃으면서 strstr을 사용하자

strstr(str1, str2)는 

str1에 str2가 존재하는 지 확인을 해준다. 근데 내부적으로 KMP를 사용하나 (확실한거아님) 백준에서 시간 초과가 되지 않는다.

```
#include <iostream>
#include <cstring>
#include <string>
#include <vector>

using namespace std;

int main()
{
	ios_base::sync_with_stdio(0);
	cin.tie(0);
	
    string str; cin >> str;
    string temp; cin >> temp;

    if (strstr(str.c_str(), temp.c_str()) != NULL) {
        cout << 1;
    }
    else {
        cout << 0;
    }
	 
}
```
