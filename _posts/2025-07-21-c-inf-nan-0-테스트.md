C++의 inf, nan에 대해서 제대로 답을 못했던 기억이 있어서 내가 직접 테스트해보면서 정리를 해보려고한다.

**공부하면서 추가로 알게 된 점**

+- 무한대, NaN의 표기법에 대해서 알게 되었다.

float를 표기할 때 , 부호 지수 가수 부분으로 나누는데

지수 부분이 255, 가수 부분이 0일 때는 INF이고,

지수 부분이 255, 가수 부분이 0이 아닐 때는 NaN이다.

### **Inf**

### inf, -inf는 같은가 다른가 / inf에 연산자를 사용할 수 있는가

log(0.0)을 통해서 -inf를 유도했다.

```
float t = std::log(0.0);
float t2 = -1 * std::log(0.0);
float t3 = 100000000 +  std::log(0.0);
```

출력되는 값은

t: -inf

t2: inf

t3: -inf

로 -를 곱해면 inf/ -inf 로 구분이 된다. 다르게 표현한다는 말은 저장방식도 다르다는 말이다.
